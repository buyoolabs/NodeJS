Comandos mas útiles de NPM:

npm search <nombre-del-modulo|palabra-clave> -> Buscar un paquete

npm info <nombre-del-modulo> -> Información del módulo

npm ls -> Módulos instalados

npm outdated -> Busca si hay versiones más nuevas acerca de los modulos instalados en el directorio actual.
